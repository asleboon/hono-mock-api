# Mock API

Simple JSON API made with Bun, Hono, zod and ts-belt.
Uses the JSONPlaceholder API for data.

## Getting Started

Install bun

```
curl -fsSL https://bun.sh/install | bash
```

```
bun install
bun run dev
```

```
open http://localhost:8080
```
